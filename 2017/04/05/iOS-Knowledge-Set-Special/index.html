【iOS知识】Set 独有的方法有哪些？

Posted on 2017-04-05 | In 知识总结 |
答答卓同学的Swift面试题

关于Set一起工作的时候貌似问过我😂
不过不是问方法，而是特殊性之类的吧。

  1. 可以去重，即[1,2,3,3]留下[1,2,3]
  2. 无序
  3. 遵循Hashable协议

答案

而独有方法的话应该是文档上说的集合操作了 文档传送门

  1. 使用intersection(_:)方法根据两个集合中都包含的值创建的一个新的集合。–>
  2. 使用symmetricDifference(_:)方法根据在一个集合中但不在两个集合中的值创建一个新的集合。
  3. 使用union(_:)方法根据两个集合的值创建一个新的集合。
  4. 使用subtracting(_:)方法根据不在该集合中的值创建一个新的集合。
